GREGIS: Greek GIS methods and data

Οι σελίδες αυτές θα φιλοξενήσουν μεθόδους και οδηγίες για την αξιοποίηση ανοιχτών γεωγραφικών δεδομένων που αφορούν την Ελλάδα με τη χρήση εργαλείων GIS ανοιχτού λογισμικού.

This project will host methods and instructions for the utilization of open geographical data for Greece using open source GIS tools.
