GREGIS: Greek GIS methods and data

Οι σελίδες αυτές φιλοξενούν μεθόδους και οδηγίες για την αξιοποίηση ανοιχτών γεωγραφικών δεδομένων που αφορούν την Ελλάδα με τη χρήση εργαλείων GIS ανοιχτού λογισμικού.

This project hosts methods and instructions for the utilization of open geographical data for Greece using open source GIS tools.

Αρχεία (files):

  • corine_grass-gis_el.pdf, Οδηγός για την "Αναπαραγωγή του χάρτη CORINE µε το GRASS-GIS"
  • clc_colors_qgis_layer_style.qml, χρώματα για το CORINE και το πρόγραμμα QGIS (stylesheet for CORINE and QGIS)
  • README
